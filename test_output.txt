Shell operators:
  ;           - Sequential execution (cmd1 ; cmd2)
  &&          - Conditional AND (cmd1 && cmd2 - run cmd2 only if cmd1 succeeds)
  ||          - Conditional OR (cmd1 || cmd2 - run cmd2 only if cmd1 fails)
  |           - Pipe output (cmd1 | cmd2 - send cmd1 output to cmd2)
  >           - Redirect output to file (cmd > file.txt)
  >>          - Append output to file (cmd >> file.txt)
  <           - Redirect input from file (cmd < file.txt)

Examples:
  ls ; pwd                     - List files then show current directory
  test -f file.txt && cat file.txt  - Show file only if it exists
  ls missing_dir || echo 'Not found'  - Show error message if ls fails
  ls | grep .txt               - List files and filter for .txt files
  echo 'Hello World' > output.txt     - Write text to file
  echo 'More text' >> output.txt      - Append text to file
